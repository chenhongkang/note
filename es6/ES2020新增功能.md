# ES2020新增功能

### 1.空值合并 （??）

空值合并(??)可以单独监测null和undefined,相对于或操作更加实用

'' ?? 1 =  ''

0 ?? 1 = 0

NaN ?? 1 = NaN

Null ?? 1 = 1

Undefined ?? 1 = 1

### 2.可选链

可选链允许访问多层嵌套的对象属性，不用担心属性是否存在

x = {b: {c:1}}

x.d.c : 报错

x.d?.c : undefined

### 3.Promise.allSettled

接受一组Promise,并且返回所有的结果，不管是resolved还是rejected
